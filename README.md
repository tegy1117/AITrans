# AI 번역 확장 프로그램

Chrome/Edge Manifest V3 기반 AI 페이지/선택 영역 번역 확장 프로그램입니다.
현 언어 한국어만 지원중이며, 나중에 다른언어 추가 예정입니다.

## 기능

- 페이지 텍스트를 번역문으로 교체하고 원문으로 복원할 수 있습니다.
- 텍스트를 선택한 뒤 우클릭 메뉴에서 "선택한 텍스트 AI로 번역"을 실행할 수 있습니다.
- 이미지 위에서 우클릭 메뉴의 "이미지 AI로 번역"을 실행할 수 있습니다.
- 페이지, 선택 영역, 일반 번역, 이미지, 사전 기능별 프롬프트 프로필을 설정할 수 있습니다.
- 여러 프로바이더를 등록하고 프로필별로 프로바이더와 모델을 선택할 수 있습니다.
- 일반 번역창에서 원하는 일반 번역 프로필을 직접 선택하고 최근 번역 기록을 저장할 수 있습니다.
- system, user, assistant 역할의 메시지를 순서대로 구성해 프리필 방식 프롬프트를 만들 수 있습니다.
- 토큰 제한, 온도, top-p, 추론 강도를 설정할 수 있습니다.
- OpenAI, Claude, Gemini, Ollama 로컬, Ollama 클라우드 API, Custom OpenAI 호환 API를 지원합니다.
- 선택/이미지 번역 결과 사이드 드로어에서 텍스트를 드래그해 여러 사전 항목을 생성하고 저장할 수 있습니다.

이미지 번역은 provider에 이미지 payload를 직접 보내는 방식입니다. 사용하는 모델은 이미지 입력을 지원하는 vision-capable 모델이어야 합니다.

## 개발

```powershell
npm.cmd install
npm.cmd test
npm.cmd run build
```

## Chrome 또는 Edge에 로드하기

1. `npm.cmd run build`를 실행합니다.
2. Chrome에서는 `chrome://extensions`, Edge에서는 `edge://extensions`를 엽니다.
3. 개발자 모드를 켭니다.
4. "Load unpacked" 또는 "압축해제된 확장 프로그램 로드"를 선택합니다.
5. `E:\AI Trans\dist` 폴더를 선택합니다.

확장 프로그램 옵션 페이지에서 프로바이더, API 키, 기본 URL, 모델, 프롬프트를 설정합니다.
Ollama 로컬 기본 URL은 `http://localhost:11434`입니다.

조만간 릴리즈 하면 zip파일로 바로 적용 가능한 파일 형태로 만들겠습니다.

## 이미지 번역 사용법

1. 옵션 페이지에서 `이미지 번역` 프로필의 provider와 vision-capable model을 설정합니다.
2. 웹페이지 이미지 위에서 우클릭합니다.
3. `이미지 AI로 번역`을 선택합니다.
4. 결과 버블에서 번역문을 복사하거나 닫을 수 있습니다.

모델이 이미지 입력을 지원하지 않으면 provider 오류가 그대로 표시됩니다.

## 일반 번역창 사용법

1. 팝업의 `일반 번역` 버튼을 누르거나 선택 번역 결과에서 `일반 번역창`을 누릅니다.
2. 옵션 페이지의 `일반 번역창 표시 방식`을 `사이드바` 또는 `새 탭`으로 선택할 수 있습니다.
3. 일반 번역창에서 사용할 일반 번역 프로필을 선택한 뒤 텍스트를 번역합니다.
4. 번역 기록에는 사용한 프로필 이름, 원문, 번역문이 함께 저장됩니다.

## 사전 기능 사용법

1. 선택 번역 또는 이미지 번역을 실행합니다.
2. 오른쪽에 열린 번역 사이드 드로어에서 사전에 추가할 텍스트를 드래그합니다.
3. 여러 텍스트를 선택한 뒤 `사전 생성`을 누릅니다.
4. 생성된 항목을 확인하고 필요한 항목만 `저장`합니다.
